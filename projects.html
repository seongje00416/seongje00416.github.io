개발 및 클라우드 프로젝트

개발 및 클라우드 구축에 대한 실제 구현 경험을 만들고 협업에 참여해보기 위해 다양한 프로젝트에 참여했습니다.

Fooding

요식업계 자영업자들을 위한 B2B 및 B2C 통합 운영 지원 플랫폼 서비스 개발

사용한 기술 스택

Spring Boot를 기반으로 한 API를 개발하고 POS, ADMIN, CUSTOMER, CEO로 나누어 RBAC 형태를 구현하고 도메인을 나누어 서비스를 개발했습니다. Redis를 사용해 조회량이 많은 메인 화면의 상점 목록에 캐싱 전략을 사용했으며 POS에서 작동하는 서비스와 사용자 앱과의 실시간 정보 일치화를 위해 Spring WebFlux를 사용해 실시간 데이터 처리를 구현했습니다.

내가 작업한 내용

  • Redis를 사용한 메인 화면의 상점 조회 서비스에 캐싱 전략 적용
  • Spring WebFlux를 사용한 POS 서비스와 USER 서비스의 실시간 통신 구현
  • Spring Boot를 사용한 Restful API 개발

경험한 것 및 배운 내용

  • Redis - 특정 데이터에 대해 Key-Value 저장소를 만들어 반복 조회가 많은 서비스에 대해서는 매번 DB에서 조회하는 것이 아닌 Redis 저장소에서 조회해 실제 DB에 몰리는 트래픽을 최소화해 데이터 조회 단계에서 사용자의 경험 개선
  • Spring WebFlux - Mono와 Flux 객체를 적당히 사용해 클라이언트와 서버 사이의 지속적인 연결 파이프라인을 만들고 서버에서 사용자가 이벤트를 발생했을 때 정보가 바로 POS에 전달되도록 해서 결제와 쿠폰 사용에 대한 정보 차이 최소화
  • Spring Boot - DB에 대한 CRUD 작업에 JPA를 활용했으며 QueryDSL을 사용해 복잡한 쿼리 조회도 무리 없이 수행할 수 있도록 구현
Spring Boot Spring WebFlux Redis Swagger

Pool

동영상 Short-Form을 활용한 소셜 네트워킹 플랫폼

사용한 기술 스택

Spring Boot 기반으로 한 API를 개발하고 소셜 네트워킹으로써 기능을 위해 채팅, 동영상 업로드, 알림 등의 기능을 포함했습니다.

내가 작업한 내용

  • gRPC를 사용한 채팅 서비스 개발
  • FCM을 사용한 실시간 알림 서비스 개발
  • Spring Boot와 JPA를 사용한 Restful API 개발
  • AWS를 사용한 클라우드 인프라 구축

경험한 것 및 배운 내용

  • gRPC - 앱 클라이언트 사이의 실시간 채팅을 위해 WebSocket보다 가볍고 빠른 gRPC를 통해 채팅 서비스 구현
  • FCM - 백그라운드에서도 알림을 받을 수 있어야 하며 토큰 기반으로 알림 서비스를 구현하기 쉬운 FCM을 통해 실시간 알림 구현
  • Spring Boot - JPA와 QueryDSL을 사용해 Restful API 구축
  • AWS - EC2를 활용해 API 서버를 구동하고 S3와 RDS를 통해 데이터를 안전하게 보관, Lambda를 사용해 S3의 동영상 업로드 기능에 대한 예외 처리
Spring Boot gRPC MySQL AWS FCM Lambda

Now & Go

AI를 사용해 사용자의 개인 맞춤형 일정을 제공하고 하나의 플랫폼에서 계획 및 결제까지 가능한 One-Stop 플랫폼 서비스 개발

사용한 기술 스택

Mana Kubernetes를 사용한 마이크로 서비스 배포 환경을 구현하고 EKS를 통해 운영하고 서비스를 관리했습니다. 클라이언트, 도메인 서비스, 데이터 서비스로 나누어진 3-Tier 구조로 통신이 이루어지며 각 서비스는 역할에 따라 Spring Boot 또는 FastAPI로 개발되었습니다.

내가 작업한 내용

  • Spring Boot를 사용한 도메인 서비스 API 개발
  • Terraform을 통한 EKS 환경 구축
  • ArgoCD를 통한 서비스 배포 자동화
  • JPA와 MyBatis를 사용한 데이터 접근 서비스 개발
  • Redis를 사용한 주문 대기열 서비스 구현
  • 토스페이먼츠 결제 API를 통한 결제 서비스 구현

경험한 것 및 배운 내용

  • Terraform을 통해 AWS에서 EKS를 운영할 수 있는 인프라 구축 및 인프라에 대한 형상 관리
  • ArgoCD와 Github Actions를 통해 개발자들이 서비스를 배포하기 쉽도록 CI/CD 파이프라인 구축
  • Redis의 SortedSet 알고리즘을 활용해 DB에 접근할 수 있는 인원을 초과했을 때 접속한 사용자에게 대기번호 부여 후 배치 서비스를 통해 사용자를 활성 큐로 옮겨 결제가 진행되도록 구현
  • 토스페이먼츠의 결제 API 모듈을 사용해 상품 구매를 위한 결제 서비스 개발
  • 엔티티에 따라 DynamoDB와 PostgreSQL을 적절히 사용하도록 구현
Spring Boot Redis Kubernetes AWS ArgoCD Github Actions DynamoDB PostgreSQL Docker

학식로그

교내 학생들이 학식을 이용하는데 정보를 쉽게 공유하고 얻을 수 있으며 쉽게 식권을 구매할 수 있는 플랫폼 서비스 개발

사용한 기술 스택

Ma Kubernetes를 사용해 마이크로 서비스를 운영할 수 있는 환경을 구축했으며 EKS를 사용해 서비스를 운영했습니다. API 서비스는 Spring Boot로 개발했으며 마이크로 서비스 사이의 데이터 정합성을 위해 Kafka를 사용했습니다.

내가 작업한 내용

  • EKS를 사용한 Kubernetes 운영 및 클러스터 구축
  • Github Actions와 ArgoCD를 통한 CI/CD 파이프라인 구축

경험한 것 및 배운 내용

  • Kubernetes를 사용한 마이크로 서비스 배포 환경 구축 및 EKS를 통해 완전 관리형 k8s 클러스터 구축
  • Github Actions와 ArgoCD를 통한 CI/CD 파이프라인을 구축해 개발 팀원들이 Kubernetes에 대한 지식이 없어도 배포할 수 있는 환경 구축
  • 운영 환경의 안정성을 위해 EC2에 자체적으로 Kubernetes 클러스터를 구축해서 멀티 스테이지 구성

사용한 기술 스택

AWS Lambda를 활용한 서버리스 API 기반으로 운영되며 DynamoDB를 통해 NoSQL 데이터베이스를 구현하고 Open AI API를 사용해 뉴스에 대한 요약 및 퀴즈를 제공합니다. WebSocket을 통해 사용자 사이에 정보를 공유하고 이야기할 수 있는 커뮤니케이션 서비스를 구현합니다.

내가 작업한 내용

  • AWS Lambda를 활용한 API 개발
  • 생성형 AI를 활용한 뉴스 요약 및 퀴즈 생성 기능 개발
  • Open API를 사용해 실시간으로 올라오는 뉴스 조회 서비스 개발
  • DynamoDB를 사용한 NoSQL 데이터베이스 구현
  • Websocket을 사용한 채팅 서비스 개발

  • 네이버 Open API를 사용해 주어진 주제에 대한 최신 뉴스 정보를 불러옴

  • Websocket을 사용해 클라이언트들이 실시간으로 데이터를 주고 받을 수 있는 환경 구축
